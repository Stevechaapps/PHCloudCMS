## How to Capture

1. Run PHCloud locally with your theme:
   ```bash
   cd ~/phcloud
   npm run dev
   ```

2. Open DevTools → Toggle Device Toolbar (Ctrl+Shift+M)

3. Select device:
   - Mobile: iPhone SE (375px)
   - Tablet: iPad (768px)
   - Desktop: Responsive (1440px)

4. Capture screenshot:
   - Chrome: Three dots → Run command → "Capture screenshot"
   - Or use extension: GoFullPage, Nimbus Screenshot

5. Save as `mobile.png`, `tablet.png`, `desktop.png` in this folder

---

**Tip:** These previews are critical for theme adoption — site owners want to see what they're installing!
